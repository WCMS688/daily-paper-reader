## 摘要
我们分析了基于PAM-4/6/8作为候选调制格式的带宽受限IM/DD系统中符号速率与码率的权衡，针对LPO收发器。采用二进制对称信道容量作为硬判决解码下的可达信息率，作为每通道200G和400G吞吐量目标的性能指标。结果表明，将FEC码率降低至KP4基线以下，可使高阶PAM格式在比PAM-4低得多的符号速率下运行，同时满足吞吐量要求。

## Abstract
We analyze the symbol rate-code rate trade-off in bandwidth-limited IM/DD systems targeting LPO transceivers, using PAM-4/6/8 as candidate modulation formats. We use the capacity of the binary symmetric channel as an achievable information rate under hard-decision decoding, serving as a performance metric for 200G and 400G per-lane throughput targets. Our results show that reducing the FEC code rate below the KP4 baseline allows higher-order PAM formats to operate at substantially lower symbol rates than PAM-4 while meeting the throughput requirement.

- **核心问题**：在面向每通道200G/400G线性驱动可插拔光学（LPO）收发器的带宽受限IM/DD系统中，如何通过调整前向纠错（FEC）码率来充分利用高阶PAM调制格式（PAM-6/8）在降低符号速率方面的优势，从而缓解带宽瓶颈和ADC/DAC功耗压力。
- **背景**：当前商业部署已实现每通道200 Gb/s的PAM-4，但扩展到400 Gb/s面临两大限制：传输链路的带宽限制和激光器相对强度噪声（RIN）。尤其在线性驱动LPO中，PCB走线等引入的频率相关损耗造成严重符号间干扰（ISI）。传统固定码率（如KP4，码率≈0.9449）评估框架下，高阶PAM因无法达到KP4的BER阈值而被认为不适合，但本文质疑这一固定码率假设，提出通过降低码率来释放高阶PAM在带宽受限场景下的潜力。

### 论文提出的方法论

- **核心思想**：放弃固定码率KP4的评估范式，转而使用**可达信息率（AIR）**作为性能指标，在可变码率下分析PAM-4/6/8的符号速率-码率权衡。核心观点是：对于给定的带宽受限信道，可以通过牺牲少量码率（即使用更强的FEC）换取显著的符号速率降低，从而提高系统整体可行性。

    - **AIR定义**：采用硬判决解码下的二进制对称信道容量公式：  
        $$\text{AIR}_{\text{HD}} = m [1 - H_2(p)]$$  
        其中 \(m = \log_2 M\)，\(p\) 为解映射后平均硬判决BER，\(H_2(p) = -p \log_2 p - (1-p)\log_2(1-p)\)。可达吞吐量 \(T_{\text{AIR}} = R_s \cdot \text{AIR}_{\text{HD}}\)，\(R_s\) 为符号速率。
    - **权衡分析**：比较不同符号速率下，为达到目标净数据速率（200G或400G）所需的码率 \(R_c^{\text{req}}\) 与信道可支持的AIR码率 \(R_c^{\text{air}}\)，找出可行工作点及码率回退裕度。

### 论文的主要结论与发现

1. **固定码率KP4误导性**：在KP4准则下，PAM-4是唯一能在200G/400G场景下达到FEC阈值的格式，而PAM-6/8完全失败，这错误地排除了高阶PAM。
2. **变码率解锁优势**：通过将FEC码率降至KP4以下（例如200G时PAM-6用0.96、PAM-8用0.92；400G时PAM-8用0.91），PAM-6/8可以在显著低于PAM-4的符号速率下满足吞吐量要求：
    - 200G：PAM-8符号速率降低28 Gbaud（相对于PAM-4的100 Gbaud降至72 Gbaud），PAM-6降低20.8 Gbaud（降至79.2 Gbaud）。
    - 400G：PAM-8降低54.1 Gbaud（200→145.9 Gbaud），PAM-6降低38.6 Gbaud（200→161.4 Gbaud）。
3. **码率回退裕度**：在可行符号速率范围内，存在一个介于所需码率与可达码率之间的“码率回退裕度”，该裕度允许实际FEC方案有非理想实现的空间，且在中低符号速率处最大。
4. **带宽扩展可行性**：即使对于45 GHz MZM（200G场景），PAM-8在降低码率后仍能实现200 Gb/s，而PAM-4不能。在400G场景，所有格式在所有测试带宽（100-120 GHz）下均可达到400 Gb/s。
